You can only use sms transactions with one registry in the current release. The registry and currency must already exist.

You also need to configure /etc/gammu-smsdrc for the phone or dongle that you are using and for inbox/outbox etc. See the example in the config sub-directory which is for name=Huawei 1750 giving an AT style connection. This is the hardware used for testing the current version.

You may also need to use the Mode Switch Utility if a phone dongle declares itself as a storage device
