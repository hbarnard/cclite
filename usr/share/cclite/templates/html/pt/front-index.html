Cclite Quickstart

Instalador da Web | Logon do usuário | Administração |
  1. Uso a análise de instalação para analisar o estado atual da instalação
  2. Uso o instalador para definir valores de configuração, banco de dados usuário e senha
  3. Criar um registro (trading group) for your system, for example, Dalston , Newington , Islington Um registro é um banco de dados mysql...
  4. Logon como gerente Logon para o seu registro como Gerenciador de com senha Gerenciador de
  5. (when Manager) Set up a currency or two
  6. Criar usuários ou deixá-los vir Criar usuários e deixá-los a negociar.
  7. Lembre-se de remover ou desabilitar o instalador e alterar as senhas de administrador e sysaccount. Remova esta página também!
Trabalhos em lote, como escrever rss, lote entrada de transação de leitura de gráficos estão agora na página de gestão para o administrador. Eles podem ser executado diretamente da web sem usar o cron. Para o cron, algumas modificações precisam ser feitas nos scripts.
