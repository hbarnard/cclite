Cclite Quickstart

Web Installer | User Logon | Administration |
  1. Use the installation analysis to analyse the current state of installation
  2. Use the installer to set up configuration values, database user and password
  3. Create a registry(trading group) for your system, for example, dalston, newington, islington A registry is one mysql database...
  4. Logon as manager Logon to your registry as manager with password manager
  5. (when Manager) Set up a currency or two
  6. Create users or let them come Create users and let them trade.
  7. Remember to remove or disable the installer and change the sysaccount and admin passwords. Remove this page also!
Batch jobs such as writing rss, reading batch transaction input and graphing are now on the management page for the administrator. They can be run straight from the web without using cron. For cron, a few modifications need to be made in the scripts.
